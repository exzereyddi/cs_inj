# injector

Инжектит любой `.dll` в `hl.exe` и затирает все следы его названия из памяти процесса.

## Что делает?

- Находит `.dll` рядом с собой
- Инжектит через `LoadLibrary + CreateRemoteThread` Метод рискрипта оставил.
- 4 прохода по памяти — затирает имя файла/полный путь
- Закрывается сам через 5 секунд

## Использование
Положи injector.exe и твой .dll в одну папку
Запусти Half-Life
Запусти injector.exe

## Пример
hackir.dll (156.4 KB)
ok
closing in 5s...
